# 《2026年湖南大学数据处理方法与技术》课程设计

上市公司AI战略言行一致性分析

## 项目简介

本项目通过爬取上市公司年报、招聘数据和专利数据，利用大语言模型提取AI战略信息，评估企业在AI战略上的"言行一致性"——即年报中宣称的AI战略与实际招聘、专利行为的匹配程度。

## 文件结构

### 代码/

| 文件 | 说明 |
|------|------|
| `上市公司信息爬取.py` | 爬取上市公司基本信息 |
| `上市公司年报爬取.py` | 爬取上市公司年度报告 |
| `PDF转文本.py` | 将年报PDF转换为可分析文本 |
| `招聘数据爬取.py` | 爬取人工智能相关招聘数据 |
| `大语言模型年报AI提取.py` | 使用大语言模型从年报中提取AI战略信息 |
| `上市公司信息清洗.py` | 清洗上市公司基本信息 |
| `年报数据清洗.py` | 清洗年报数据 |
| `人工智能招聘数据清洗代码.py` | 清洗AI招聘数据 |
